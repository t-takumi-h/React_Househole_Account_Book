# 家計簿アプリ

## 目的

React の勉強のために家計簿アプリを作成しています。

## 動作確認方法(フロント側)

1. `git clone https://github.com/t-takumi-h/React_Househole_Account_Book.git`
2. `cd ./React_Househole_Account_Book/front`
3. `yarn install`
4. `yarn start`

## 動作確認方法(バッグ側)
1. `git clone https://github.com/t-takumi-h/React_Househole_Account_Book.git`
2. `cd ./React_Househole_Account_Book`
3. `docker-compose up -d`
